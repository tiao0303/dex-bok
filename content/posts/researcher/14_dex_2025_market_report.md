## 二、Uniswap V4：革命性升级

### 2.1 核心创新

Uniswap V4于2025年2月正式发布，是迄今为止最大规模的协议升级：

**三大突破：**

1. **Hooks（钩子）机制**
   - 允许开发者在流动性池生命周期中插入自定义逻辑
   - 应用场景：动态手续费、limit orders、时间加权平均价格（TWAP）预言机、range orders
   - 代码示例：
   ```solidity
   struct Hook {
       address hook;           // 钩子合约地址
       uint256 beforeSwapDelta; // swap前的钩子回调
       uint256 afterSwapDelta;  // swap后的钩子回调
   }
   ```

2. **Singleton架构**
   - 将所有代币对合并到单一合约，大幅降低Gas成本
   - 部署成本降低约90%

3. **增强型预言机**
   - TWAP计算精度提升，支持更复杂的金融产品

### 2.2 对市场的影响

- V4发布后，Uniswap市场份额从32%提升至35.9%
- 流动性提供者（LP）收益提升约15-20%
- 第三方协议基于V4 Hook开发的新产品涌现
