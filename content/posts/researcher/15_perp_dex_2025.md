## 二、Hyperliquid：重新定义Perp DEX

### 2.1 核心数据

Hyperliquid在2025年的数据堪称炸裂：

- **累计交易量：** $3万亿美元（超过大多数CEX）
- **年利润：** $12亿美元
- **代币回购：** 2025年累计回购$6.45亿HYPE代币
- **日均交易量：** 经常突破$100亿
- **2025年7月：** 现货+永续合计$3308亿，超越Robinhood ($2378亿) 39%

### 2.2 技术架构

Hyperliquid的核心创新在于**HyperCore**——一条专为交易设计的Layer1区块链：

```
HyperCore 特性：
├── 亚秒级结算 (sub-second settlement)
├── 零Gas费 (zero gas fees)
├── 20万TPS吞吐量 (200,000 TPS)
├── 订单簿在链上 (on-chain orderbook)
└── 订单延迟 < 1ms
```

**关键创新：HIPO（Hyperliquid Initial Perpetual Offering）**

2025年Hyperliquid通过HIPO发行机制，在自有链上发行了多个代币，建立了完整的一级市场发行生态。

### 2.3 USDH 稳定币

Hyperliquid发行的USDH（超额抵押稳定币）成为其生态核心：

- 抵押率：最低 110%
- 应用场景：交易保证金、流动性提供、跨协议借贷
- 2025年Q4，USDH流通量突破$8亿

### 2.4 HIP-3 提案

Hyperliquid的HIP-3提案允许第三方在HyperEVM上部署智能合约，将Hyperliquid从单一应用扩展为**应用链生态系统**。

## 三、竞争格局：Aster的挑战与Lighter的差异化

### 3.1 Aster：短暂超越

Aster在2025年8-9月一度超越Hyperliquid，占据Perp DEX市场份额第一：

- **技术路线：** 基于Solana代码库的Layer2
- **竞争优势：** Solana生态深度集成、低延迟
- **弱点：** 流动性不如Hyperliquid深厚

Aster的短暂领先证明了Perp DEX市场尚未形成垄断格局，竞争仍在继续。

### 3.2 Lighter：订单簿专业化

Lighter走了一条差异化路线：

- **专注BTC永续：** BTC相关交易对占总交易量85%+
- **订单簿模式：** 介于CEX和AMM之间
- **2025年增长：** 从市场新人到7.2%份额

### 3.3 GMX：多链布局

GMX通过Arbitrum和Avalanche双线布局：

- **GLP金库模型：** 流动性提供者承担对手盘风险
- **2025年升级：** GLP 2.0引入动态做市商机制
- **多链优势：** 捕获各链原生资产交易需求

### 3.4 dYdX：v5升级完成

dYdX于2025年完成v5升级：

- 从StarkWare迁移到自有链（dYdX Chain）
- 完全去中心化的验证者网络
- 订单簿模式回归，支持更复杂订单类型

## 四、AMM vs 订单簿：技术路线之争

Perp DEX存在两种核心技术路线：

### 4.1 AMM模式（Hyperliquid、GMX）

```
交易者多头/空头 ← → 流动性池（LP做市）
```

**优势：**
- 流动性即时可用
- 无需对手方
- 合约部署简单

**劣势：**
- 无常损失风险（LP）
- 大额交易滑点高

### 4.2 订单簿模式（dYdX、Lighter）

```
买入订单 ← → 订单簿 ← → 卖出订单
```

**优势：**
- 机构级交易体验
- 滑点接近零
- 支持复杂订单类型

**劣势：**
- 需要足够的做市商参与
- 前端复杂性高

### 4.3 2025年的趋势

2025年，两种路线出现**融合趋势**：
- GMX引入"vAMM"概念，保留LP效率的同时接近订单簿价格发现
- dYdX Chain支持LP注入流动性，降低做市商门槛
- Hyperliquid推出"限价单簿"，增加订单簿功能

## 六、普通交易者指南

1. **追求深度流动性** → Hyperliquid
2. **偏好订单簿体验** → dYdX
