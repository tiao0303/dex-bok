## DeFi 收益来源

### 1. 交易手续费 ⭐⭐⭐⭐⭐

**来源**：用户 swap 支付的费用
**分配**：按 LP 份额比例分配
**特点**：
- 最稳定
- 与市场波动无关
- APY 5-50%

**代表**：Uniswap、Curve、SushiSwap

### 2. 流动性挖矿奖励 ⭐⭐⭐⭐

**来源**：协议代币排放
**分配**：按 LP 份额 + 乘数
**特点**：
- 收益高但不稳定
- 代币价格波动大
- 早期项目风险高

**代表**：Uniswap (UNI)、Curve (CRV)、Balancer (BAL)

### 3. 借贷利息 ⭐⭐⭐⭐

**来源**：借款人支付利息
**分配**：按存款比例
**特点**：
- 相对稳定
- 单币存入（无 IL）
- APY 2-20%

**代表**：Aave、Compound、MakerDAO

### 4. 质押奖励 ⭐⭐⭐

**来源**：PoS 网络验证奖励
**分配**：按质押量
**特点**：
- 无 IL
- 锁定期限制
- APY 3-15%

**代表**：ETH2.0、Lido、Rocket Pool

## 热门 Farm 策略

### 1. Curve 3pool + Convex ⭐⭐⭐⭐⭐

**流程**：
```
1. 存入 USDC/USDT/DAI 到 Curve 3pool
2. 获得 3CRV LP Token
3. 质押到 Convex
4. 获得 CRV + CVX 双重奖励
```

**收益**：
- 基础 APY：3-5%
- CRV 奖励：2-4%
- CVX 奖励：2-4%
- **总计**：7-13%

**风险**：低（稳定币）

### 2. ETH/stETH ⭐⭐⭐⭐

**流程**：
```
1. ETH 质押到 Lido → stETH
2. 存入 Curve stETH/ETH 池
3. 获得 LP Token
4. 质押到 Convex 获取额外奖励
```

**收益**：
- stETH 质押：3-4%
- Curve 手续费：1-2%
- CRV/CVX奖励：2-5%
- **总计**：6-11%

**风险**：中（stETH 脱锚风险）

### 3. Arbitrum GMX ⭐⭐⭐⭐

**流程**：
```
1. 购买 GMX 代币
2. 质押获取 escrowed GMX
3. 获得 ETH 分红
```

**收益**：
- ETH 分红：5-15%
- 代币增值：不确定
- **总计**：10-30%+

**风险**：中高（代币价格波动）

### 4. Pendle 固定收益 ⭐⭐⭐

**创新**：
- 将收益 token 化
- 可交易未来收益
- 锁定固定 APY

**策略**：
```
1. 购买 PT（本金 token）
2. 到期赎回本金
3. 或卖出 YT（收益 token）
```

**收益**：固定 5-15%

**风险**：中（新协议）

## 风险详解

### 1. 智能合约风险 ⭐⭐⭐⭐⭐

**案例**：
- 2022 Nomad 桥被盗 $1.9 亿
- 2021 Cream Finance 被盗 $1.3 亿

**防护**：
- 只用经过审计的协议
- 查看审计公司（OpenZeppelin、Trail of Bits）
- 小额测试

### 2. 脱锚风险 ⭐⭐⭐⭐

**案例**：
- 2022 UST 脱锚 → 归零
- 2023 USDC 短暂脱锚 → $0.88

**防护**：
- 分散多个稳定币
- 避免算法稳定币
- 关注储备金报告

### 3. 代币价格风险 ⭐⭐⭐⭐

**问题**：
- 挖矿奖励代币暴跌
- APY 再高也亏钱

**防护**：
- 及时卖出奖励代币
- 不要复投所有奖励
- 计算真实收益（USD 计）

## 收益优化技巧

### 1. 复投策略

| 频率 | 优点 | 缺点 |
|------|------|------|
| 每日 | 收益最大化 | Gas 费高 |
| 每周 | 平衡 | 适中 |
| 每月 | Gas 费低 | 收益损失 |

**建议**：
- > $10k：每日复投
- $1-10k：每周复投
- < $1k：每月复投

### 2. Gas 优化

- 用 Layer2（Arbitrum、Optimism）
- 选择 Gas 低时段（亚洲凌晨）
- 批量操作（一次多笔）

### 3. 奖励代币处理

**保守**：全部卖出 → 锁定收益
**平衡**：50% 卖出 + 50% 复投
**激进**：全部复投 → 博代币增值

## 总结

DeFi 收益 farming 核心原则：

1. **风险第一**：高 APY 往往高风险
2. **分散配置**：不要 All-in 一个策略
3. **及时止盈**：奖励代币及时卖出
4. **持续学习**：新协议不断出现

**记住**：保住本金比追求高收益更重要。
