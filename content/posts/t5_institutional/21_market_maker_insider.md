## 什么是专业做市商

- 赚取买卖价差（Spread）
- 对冲方向性风险

## 做市策略详解

### 策略一：网格做市 ⭐⭐⭐⭐⭐

#### 原理
```
在当前价格上下设置多层买单和卖单

示例（ETH $1800）：
卖单：$1810, $1820, $1830, $1840
买单：$1790, $1780, $1770, $1760

每层成交后自动补单
```

#### 收益来源
- 买卖价差：0.5-1%
- 交易返佣：0.01-0.05%
- 套利收益：跨所/跨链

#### 参数设置
| 参数 | 保守 | 激进 |
|------|------|------|
| 网格层数 | 10 层 | 50 层 |
| 每层间距 | 1% | 0.2% |
| 单层金额 | 总资金 5% | 总资金 1% |
| 适用场景 | 震荡市 | 高波动 |

#### 风险
- 单边行情库存风险
- 需要频繁调仓
- Gas 成本高（链上）

---

### 策略二：统计套利 ⭐⭐⭐⭐

#### 原理
利用历史统计规律，寻找价格偏离

**常见策略**：
```
1. 期现套利
   - 期货价格 > 现货 → 卖期货买现货
   - 等待收敛平仓

2. 跨所套利
   - Binance 价格 > FTX → 卖 Binance 买 FTX
   - 等待价差收敛

3. 配对交易
   - ETH/BTC 比率偏离历史均值
   - 做空高估，做多低估
```

#### 技术要求
- 历史数据回测
- 实时价格监控
- 自动化执行系统

---

### 策略三：Delta 中性 ⭐⭐⭐⭐⭐

#### 原理
```
构建 Delta=0 的组合
- 方向性风险为零
- 只赚取其他收益（费率、波动率）
```

#### 实战案例
**场景**：ETH $1800

**操作**：
```
1. 买入 10 ETH 现货（Delta +10）
2. 卖出 10 ETH 永续合约（Delta -10）
3. 净 Delta = 0

收益来源：
- 资金费率：年化 5-20%
- 无方向性风险
```

#### 优化
```
进阶玩法：
- 多币种分散
- 跨平台资金费率套利
- 动态调整对冲比例
```

## 机构技术栈

### 1. 低延迟基础设施
| 组件 | 要求 | 成本 |
|------|------|------|
| 服务器 | 托管在交易所机房 | $10k/月 |
| 网络 | 光纤直连 | $5k/月 |
| 数据库 | 内存数据库（Redis） | $2k/月 |
| 编程语言 | C++/Rust/Go | - |

### 2. 风险管理系统
```
实时监控：
- 总敞口
- 单一资产风险
- 保证金水平
- 异常交易检测

自动风控：
- 止损触发
- 仓位限制
- 紧急平仓
```

### 3. 执行算法
| 算法 | 用途 | 特点 |
|------|------|------|
| TWAP | 大额拆单 | 时间加权平均 |
| VWAP | 跟随成交量 | 成交量加权平均 |
| Iceberg | 隐藏大单 | 只显示部分 |
| Sniper | 抢跑/抢单 | 极低延迟 |

## DEX 做市 vs CEX 做市

### DEX 做市特点
| 优势 | 劣势 |
|------|------|
| 无需 KYC | Gas 费高 |
| 7x24 可访问 | 速度慢（区块确认） |
| 透明度高 | MEV 风险 |
| 创新策略多 | 智能合约风险 |

### CEX 做市特点
| 优势 | 劣势 |
|------|------|
| 速度快（微秒级） | 需要 KYC/审核 |
| 费用低 | 不透明 |
| 流动性深 | 交易所风险 |
| 专业工具 | 门槛高 |

## 向机构学习

### 散户可借鉴的

#### 1. 风险管理
```
机构做法：
- 单笔风险<总资金 1%
- 最大回撤<10%
- 严格止损

散户应用：
- 设置止损线
- 不要 All-in
- 定期复盘
```

#### 2. 策略系统化
```
机构做法：
- 策略回测
- 参数优化
- 实盘验证

散户应用：
- 记录交易历史
- 分析胜率
- 优化策略
```

#### 3. 分散投资
```
机构做法：
- 多策略并行
- 多资产配置
- 多平台分散

散户应用：
- 不要只做一个池子
- 稳定币 + 主流币 + 小仓位博弈
- 多个 DEX 分散
```

### 散户做不到的（不必强求）

- ❌ 高频交易（速度劣势）
- ❌ 复杂衍生品（门槛高）
- ❌ 大额套利（Gas 费吃掉）
- ❌ 信息优势（机构有内部渠道）
