## DeFi 税务基础

### 应税事件
| 行为 | 是否应税 | 说明 |
|------|----------|------|
| 买入加密货币 | 否 | 法币→crypto |
| 卖出加密货币 | 是 | crypto→法币 |
| Swap 交易 | 是 | crypto→crypto |
| 提供流动性 | 可能 | 视司法管辖区 |
| 质押奖励 | 是 | 视为收入 |
| 空投 | 是 | 收到时计税 |

## 主要国家/地区

### 美国（IRS）
- 资本利得税：短期 10-37%，长期 0-20%
- 收入税：挖矿/质押奖励按收到时市价
- 表格：Form 8949 + Schedule D

### 中国
- 加密货币交易：目前政策不明朗
- 建议：咨询专业税务师

### 新加坡
- 长期持有：免税
- 交易收入：按个人所得税

## 记录保存

### 必需信息
- 交易日期和时间
- 交易对和数量
- 当时 USD 价值
- Gas 费
- 钱包地址

### 建议做法
1. 使用组合追踪工具（DeBank、Zapper）
2. 导出所有交易所历史记录
3. 保留所有交易哈希
4. 定期备份记录
