## 工具二：Dune Analytics ⭐⭐⭐⭐⭐

### 什么是 Dune
- 链上数据 SQL 查询平台
- 社区创建 Dashboard
- 免费使用

### 核心功能

#### 1. 查询数据
```sql
-- 示例：查询 Uniswap V3 日交易量
SELECT 
  DATE_TRUNC('day', evt_block_time) AS day,
  SUM(amount_usd) AS volume
FROM uniswap_v3_ethereum.evt_swap
GROUP BY 1
ORDER BY 1 DESC
LIMIT 30
```

#### 2. 创建 Dashboard
- 可视化展示
- 实时更新
- 分享链接

#### 3. 订阅警报
- 价格突破

- TVL 变化

## 工具五：Token Terminal ⭐⭐⭐⭐

### 专注方向
- 协议收入分析
- 财务指标对比
- 估值模型

### 核心指标

#### 1. 收入指标
| 指标 | 说明 | 用途 |
|------|------|------|
| Revenue | 协议收入 | 盈利能力 |
| Fees | 用户付费 | 需求强度 |
| Token Emission | 代币释放 | 通胀压力 |

#### 2. 估值指标
| 指标 | 公式 | 用途 |
|------|------|------|
| P/S | 市值/收入 | 估值高低 |
| P/E | 市值/利润 | 回本周期 |
| FDV/TVL | 全流通/锁仓 | 泡沫程度 |

### 实战应用
```
筛选低估协议：
1. P/S < 5
2. 收入增长 >50%
3. TVL 稳定增长
4. 代币释放少
```
