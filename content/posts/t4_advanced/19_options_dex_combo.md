## 为什么需要期权对冲

### LP 的痛点
| 问题 | 影响 | 期权解决方案 |
|------|------|--------------|
| 无常损失 | 币价波动亏损 | Put 期权保护 |
| 踏空风险 | 币价上涨少赚 | Call 期权参与 |
| 收益不稳定 | APY 波动大 | 期权权利金收入 |

---

## 策略一：LP+Put 期权保护 ⭐⭐⭐⭐⭐

### 原理
```
持有：ETH + USDC 做 LP
买入：ETH Put 期权（行权价低于当前价）

效果：
- ETH 下跌：Put 期权盈利，对冲 LP 损失
- ETH 上涨：LP 收益增加，仅损失权利金
```

### 实战案例
**场景**：ETH = $1800，做 LP $10,000

**操作**：
```
1. $5,000 ETH + $5,000 USDC → Uniswap LP
2. 买入 ETH Put 期权
   - 行权价：$1600
   - 到期：30 天
   - 权利金：2% ($100)
```

**结果对比**：
| ETH 价格 | 无对冲 LP | 有对冲 LP |
|----------|-----------|-----------|
| $1500 (-17%) | $8,500 | $9,400 ✅ |
| $1800 (0%) | $10,000 | $9,900 |
| $2100 (+17%) | $11,200 | $11,100 ✅ |

### 推荐平台
- **Ribbon Finance**：自动化对冲
- **Hegic**：自定义期权
- **Lyra**：低费用

## 策略二：Covered Call 增强收益 ⭐⭐⭐⭐

### 原理
```
持有：ETH
卖出：Call 期权（行权价高于当前价）
收入：权利金
```

### 实战案例
**场景**：持有 10 ETH，当前$1800

**操作**：
```
卖出 Call 期权：
- 行权价：$2000 (+11%)
- 到期：30 天
- 权利金：3% ($540)
```

**结果**：
| ETH 价格 | 仅持有 | Covered Call |
|----------|--------|--------------|
| $1700 | $17,000 | $17,540 ✅ |
| $1900 | $19,000 | $19,540 ✅ |
| $2100 | $21,000 | $20,540 (被行权) |

### 优点
- ✅ 额外收入 2-5%/月
- ✅ 熊市保护
- ✅ 适合震荡市

### 缺点
- ❌ 上涨空间有限
- ❌ 大涨时踏空

### 适用场景
- 震荡市（$1600-2000 区间）
- 不指望大涨
- 追求稳定收益

## 策略三：Protective Put 保底 ⭐⭐⭐⭐

### 原理
```
持有：ETH
买入：Put 期权（行权价低于当前价）
效果：锁定最低卖出价
```

### 实战案例
**场景**：持有 10 ETH @ $1800，担心下跌

**操作**：
```
买入 Put 期权：
- 行权价：$1600 (-11%)
- 到期：60 天
- 权利金：4% ($720)
```

**保底效果**：
| ETH 价格 | 仅持有 | Protective Put |
|----------|--------|----------------|
| $1400 | $14,000 | $15,280 ✅ |
| $1600 | $16,000 | $15,280 |
| $1800 | $18,000 | $17,280 |
| $2000 | $20,000 | $19,280 ✅ |

**最大损失**：锁定在$15,280（无论跌多少）

### 适用场景
- 熊市预期
- 重大事件前（如 FOMC）
- 不想卖出但怕暴跌

## 策略四：Collar 零成本对冲 ⭐⭐⭐

### 原理
```
持有：ETH
卖出：Call 期权（高行权价）
买入：Put 期权（低行权价）

目标：权利金收支相抵
```

### 实战案例
**场景**：持有 10 ETH @ $1800

**操作**：
```
卖出 Call: 行权价$2000，收入$500
买入 Put: 行权价$1600，支出$500
净成本：$0
```

**结果**：
| ETH 价格 | 仅持有 | Collar |
|----------|--------|--------|
| $1400 | $14,000 | $15,500 ✅ |
| $1600 | $16,000 | $15,500 |
| $1800 | $18,000 | $17,500 |
| $2000 | $20,000 | $19,500 |
| $2200 | $22,000 | $19,500 (封顶) |

### 优点
- ✅ 零成本对冲
- ✅ 下跌有保护
- ✅ 适合长期持有

### 缺点
- ❌ 上涨收益封顶
- ❌ 需要期权知识

## 策略五：LP+ 期权组合 ⭐⭐⭐⭐⭐

### 完整策略
```
资金分配：
- 60% DEX LP (赚取手续费)
- 20% ETH 持有 (长期)
- 10% Put 期权 (对冲 LP 风险)
- 10% Covered Call (增强收益)
```

### 预期收益
| 来源 | 预期 APY |
|------|----------|
| LP 手续费 | 15% |
| LP 奖励 | 5% |
| Covered Call | 10% |
| 期权对冲成本 | -3% |
| **净 APY** | **27%** |

### 风险控制
- 最大回撤：<15%
- 夏普比率：>2
- 胜率：>70%

## 期权入门建议

### 新手起步
1. 先学习期权基础（行权价、到期日等）
2. 小额试水（<$1000）
3. 从 Covered Call 开始（最简单）
4. 逐步学习复杂策略

### 风险管理
- 期权仓位<总资金 20%
- 单一策略<10%
- 设置止损线
- 定期复盘
