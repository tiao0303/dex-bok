## 什么是套利

**套利** = 利用价格差异，低买高卖，赚取无风险收益

### 套利特点
- ✅ 理论上无风险
- ✅ 收益即时锁定
- ⚠️ 需要快速执行
- ⚠️ Gas 费可能吃掉利润

---

## 策略一：三角套利 ⭐⭐⭐⭐

### 原理
```
USDC → ETH → WBTC → USDC
     ↑              ↓
     └──── 利润 ←───┘
```

### 案例演示
**假设价格**：
- USDC/ETH = 1800
- ETH/WBTC = 0.065
- WBTC/USDC = 28500

**计算**：
```
1. 10000 USDC → 5.55 ETH
2. 5.55 ETH → 0.36075 WBTC
3. 0.36075 WBTC → 10281 USDC
4. 利润：281 USDC (2.81%)
```

### 执行要点
1. **速度第一**：用 Flashbots 避免抢跑
2. **Gas 优化**：选择低峰时段
3. **滑点控制**：设置合理上限
4. **监控工具**：自建价格监控脚本

### 风险
- 价格快速变化
- 交易失败损失 Gas
- MEV 机器人竞争

## 策略二：跨所套利 ⭐⭐⭐⭐

### 原理
同一资产在不同 DEX 价格不同

**案例**：
| DEX | ETH 价格 |
|-----|----------|
| Uniswap | $1800 |
| Sushiswap | $1810 |
| Curve | $1795 |

**操作**：
```
1. Curve 买入 ETH @ $1795
2. Sushiswap 卖出 ETH @ $1810
3. 利润：$15/ETH (0.83%)
```

### 推荐工具
- **1inch**：自动发现最优价格
- **Matcha**：多 DEX 对比
- **自建监控**：实时扫描价差

### 注意事项
- 考虑提现时间和 Gas
- 大额可能影响价格
- 流动性深度要足够

## 策略三：闪电贷套利 ⭐⭐⭐⭐⭐

### 什么是闪电贷
- 无需抵押
- 同一区块内借还
- 违约则交易回滚

### 平台对比
| 平台 | 支持资产 | 费用 |
|------|----------|------|
| Aave | 多种主流 | 0.09% |
| dYdX | ETH/稳定币 | 0 |
| Uniswap | 任意 Token | 0.3% |

### 经典案例
```
1. 闪电贷 1000 万 USDC
2. 低价买入被低估资产
3. 高价卖出
4. 归还贷款 + 费用
5. 利润落袋
```

### 技术要求
- 智能合约编程能力
- 理解区块打包机制
- 熟悉 Flashbots

## 策略四：流动性清算套利 ⭐⭐⭐

### 原理
监控借贷协议清算线，抢先清算获利

**平台**：
- Aave
- Compound
- MakerDAO

**收益**：
- 清算奖励：5-10%
- 需要监控 + 快速执行

### 工具
- **DefiSaver**：自动清算
- **自建机器人**：监控健康因子
