## 什么是脱锚

**脱锚** = 稳定币价格偏离 1 USD

| 程度 | 价格范围 | 风险等级 |
|------|----------|----------|
| 正常波动 | $0.99-1.01 | 🟢 低 |
| 轻度脱锚 | $0.95-0.99 | 🟡 中 |
| 严重脱锚 | $0.80-0.95 | 🟠 高 |
| 崩盘 | <$0.80 | 🔴 极高 |

## 案例一：UST 崩盘 (2022.05) 🔴

### 事件回顾
```
2022.05.02: UST = $0.998 (正常)
2022.05.05: UST = $0.985 (开始脱锚)
2022.05.07: UST = $0.65  (恐慌性抛售)
2022.05.09: UST = $0.30  (彻底崩盘)
2022.05.12: UST = $0.10  (归零)
```

### 损失统计
| 项目 | 损失 |
|------|------|
| 市值蒸发 | $400 亿 → $1 亿 |
| LUNA 归零 | $300 亿 → $0 |
| 用户损失 | 超 200 万人 |

### 崩盘原因
1. **算法机制缺陷**
   - 依赖 LUNA 铸造/销毁
   - 死亡螺旋无法避免

2. **锚定池不足**
   - 3.5 亿 UST vs 180 亿流通
   - 挤兑时无法兑付

3. **外部攻击**
   -  Curve 池子被大额砸盘
   -  引发恐慌性抛售

4. **信心崩溃**
   - 大户率先出逃
   - 散户跟风踩踏

### 教训
- ❌ 算法稳定币风险极高
- ❌ 不要相信"永远锚定"
- ✅ 只使用有储备的稳定币
- ✅ 分散多个稳定币

## 案例二：USDC 短暂脱锚 (2023.03) 🟡

### 事件回顾
```
2023.03.10: 硅谷银行倒闭
2023.03.11: USDC = $0.95 (恐慌)
2023.03.12: USDC = $0.88 (最低点)
2023.03.13: 美联储介入
2023.03.14: USDC = $0.97 (恢复)
2023.03.15: USDC = $1.00 (完全恢复)
```

### 脱锚原因
- Circle 有 $33 亿存在硅谷银行
- 占储备金 8%
- 市场担心无法兑付

### 为什么能恢复
1. **足额储备**：92% 现金 + 国债
2. **政府兜底**：美联储承诺兑付
3. **透明度高**：定期公布储备报告
4. **机构信任**：BlackRock 等背书

### 教训
- ✅ 选择有足额储备的稳定币
- ✅ 关注储备金构成
- ✅ 分散风险（不要 All-in 一个）
- ⚠️ 即使是 USDC 也有风险

## 案例三：BUSD 被叫停 (2023.02) 🟠

### 事件回顾
```
2023.02.13: 纽约监管机构要求 Paxos 停止发行 BUSD
2023.02.14: BUSD = $0.99 (轻度脱锚)
2023.02.15: 币安宣布支持兑换
2023.02.20: BUSD = $1.00 (恢复)
```

### 原因
- SEC 指控 BUSD 是未注册证券
- Paxos 被迫停止发行
- 币安逐步退出

### 结果
- 用户可 1:1 兑换
- 无实际损失
- 但流动性逐渐枯竭

### 教训
- ⚠️ 监管风险是真实存在的
- ⚠️ 中心化稳定币有政策风险
- ✅ 关注监管动态
- ✅ 分散使用多个稳定币

## 案例四：FRAX 保卫战 (2023.03) 🟢

### 事件背景
- FRAX 是部分抵押算法稳定币
- 抵押率约 90% + 10% 算法

### 脱锚情况
```
2023.03.11: FRAX = $0.97 (硅谷银行事件波及)
2023.03.12: FRAX = $0.95 (最低点)
2023.03.13: FRAX = $0.98 (快速恢复)
2023.03.14: FRAX = $1.00 (完全恢复)
```

### 为什么能恢复
1. **高抵押率**：90%+ 足额储备
2. **快速反应**：增加抵押品
3. **社区信心**：FXS 持有者支持
4. **透明度高**：实时公布储备

### 教训
- ✅ 部分抵押比纯算法更安全
- ✅ 高抵押率是关键
- ✅ 透明度建立信心

## 稳定币风险评估框架

### 1. 储备金分析 ⭐⭐⭐⭐⭐
| 指标 | 安全 | 危险 |
|------|------|------|
| 抵押率 | >95% | <80% |
| 现金比例 | >50% | <20% |
| 国债比例 | >30% | <10% |
| 商业票据 | <10% | >30% |

### 2. 发行方透明度 ⭐⭐⭐⭐
- ✅ 定期审计报告
- ✅ 实时储备查询
- ✅ 知名审计机构
- ❌ 黑箱操作

### 3. 去中心化程度 ⭐⭐⭐
| 类型 | 例子 | 风险 |
|------|------|------|
| 中心化 | USDC、BUSD | 监管风险 |
| 去中心化 | DAI、FRAX | 智能合约风险 |
| 算法 | (已崩盘的 UST) | 极高风险 ❌ |

### 4. 流动性深度 ⭐⭐⭐⭐
- 日交易量 >$1 亿
- 支持主流 DEX
- 多链部署

## 脱锚预警信号

### 红色警报 🚨
- 储备金报告逾期
- 高管离职
- 监管调查新闻
- 大额转账到交易所

### 黄色警报 ⚠️
- 价格持续<$0.99
- DEX 池子不平衡
- 社交媒体恐慌情绪
- 交易量异常放大

### 应对策略
```
黄色警报：
- 减仓 50%
- 准备兑换

红色警报：
- 立即全部兑换
- 不要等恢复
```

## 总结

稳定币脱锚教训：
1. **没有绝对安全**：USDC 也会脱锚
2. **分散是关键**：不要 All-in 一个
3. **关注储备**：定期看报告
4. **警惕算法**：纯算法=高风险
5. **快速反应**：发现警报立即行动
