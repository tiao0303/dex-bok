流动性管理策略 - 从被动 LP 到主动管理

三种 LP 策略对比

策略难度收益潜力风险适合人群
被动 LP⭐⭐⭐⭐⭐新手
主动调仓⭐⭐⭐⭐⭐⭐⭐⭐⭐进阶
集中流动性⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐专业

策略一：被动 LP

操作方式

  1. 存入 50:50 双币
  2. 获得 LP Token
  3. 坐收手续费，不做任何操作

优点

  • 简单，一次设置即可
  • 无需关注市场波动
  • 适合长期看好双币

缺点

  • 资金效率低
  • 无常损失完全暴露
  • APY 通常较低（5-15%）

推荐场景

  • 稳定币对（USDC/DAI）
  • 强相关资产（ETH/stETH）
  • 长期持有信念强

策略二：主动调仓

核心思路

定期 rebalance 回 50:50 比例，降低无常损失。

操作步骤

1. 监控池子比例
2. 当偏离>20% 时触发
3. 卖出多的，买入少的
4. 重新存入池子

优点

  • 降低无常损失 30-50%
  • 可在震荡市获利
  • 灵活性高

缺点

  • 需要频繁操作
  • 产生额外 gas 费
  • 可能错过单边行情

工具推荐

  • DeFi Saver：自动 rebalance
  • Gelato：条件触发执行
  • 自建脚本监控

策略三：集中流动性（Uniswap V3）

革命性创新

V3 允许自定义价格区间，资金效率提升 10-100 倍。
