DeFi 收益来源

1. 交易手续费 ⭐⭐⭐⭐⭐

来源：用户 swap 支付的费用 分配：按 LP 份额比例分配 特点：

  • 最稳定
  • 与市场波动无关
  • APY 5-50%

代表：Uniswap、Curve、SushiSwap

2. 流动性挖矿奖励 ⭐⭐⭐⭐

来源：协议代币排放 分配：按 LP 份额 + 乘数 特点：

  • 收益高但不稳定
  • 代币价格波动大
  • 早期项目风险高

代表：Uniswap (UNI)、Curve (CRV)、Balancer (BAL)

3. 借贷利息 ⭐⭐⭐⭐

来源：借款人支付利息 分配：按存款比例 特点：

  • 相对稳定
  • 单币存入（无 IL）
  • APY 2-20%

代表：Aave、Compound、MakerDAO

4. 质押奖励 ⭐⭐⭐

来源：PoS 网络验证奖励 分配：按质押量 特点：

  • 无 IL
  • 锁定期限制
  • APY 3-15%

代表：ETH2.0、Lido、Rocket Pool
