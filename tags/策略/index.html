做市策略详解

策略一：网格做市 ⭐⭐⭐⭐⭐

原理

在当前价格上下设置多层买单和卖单

示例（ETH $1800）：
卖单：$1810, $1820, $1830, $1840
买单：$1790, $1780, $1770, $1760

每层成交后自动补单

收益来源

  • 买卖价差：0.5-1%
  • 交易返佣：0.01-0.05%
  • 套利收益：跨所/跨链

参数设置

参数保守激进
网格层数10 层50 层
每层间距1%0.2%
单层金额总资金 5%总资金 1%
适用场景震荡市高波动

风险

  • 单边行情库存风险
  • 需要频繁调仓
  • Gas 成本高（链上）

期权+DEX 组合策略 - 对冲风险增强收益

为什么需要期权对冲

LP 的痛点

问题影响期权解决方案
无常损失币价波动亏损Put 期权保护
踏空风险币价上涨少赚Call 期权参与
收益不稳定APY 波动大期权权利金收入

策略一：LP+Put 期权保护 ⭐⭐⭐⭐⭐

原理

持有：ETH + USDC 做 LP
买入：ETH Put 期权（行权价低于当前价）

效果：
- ETH 下跌：Put 期权盈利，对冲 LP 损失
- ETH 上涨：LP 收益增加，仅损失权利金

实战案例

场景：ETH = $1800，做 LP $10,000

操作：

1. $5,000 ETH + $5,000 USDC → Uniswap LP
2. 买入 ETH Put 期权
   - 行权价：$1600
   - 到期：30 天
   - 权利金：2% ($100)

结果对比：

ETH 价格无对冲 LP有对冲 LP
$1500 (-17%)$8,500$9,400 ✅
$1800 (0%)$10,000$9,900
$2100 (+17%)$11,200$11,100 ✅

推荐平台

  • Ribbon Finance：自动化对冲
  • Hegic：自定义期权
  • Lyra：低费用

DEX 套利策略详解 - 从零开始赚无风险收益

什么是套利

套利 = 利用价格差异，低买高卖，赚取无风险收益

套利特点

  • ✅ 理论上无风险
  • ✅ 收益即时锁定
  • ⚠️ 需要快速执行
  • ⚠️ Gas 费可能吃掉利润

策略一：三角套利 ⭐⭐⭐⭐

原理

USDC → ETH → WBTC → USDC
     ↑              ↓
     └──── 利润 ←───┘

案例演示

假设价格：

  • USDC/ETH = 1800
  • ETH/WBTC = 0.065
  • WBTC/USDC = 28500

计算：

1. 10000 USDC → 5.55 ETH
2. 5.55 ETH → 0.36075 WBTC
3. 0.36075 WBTC → 10281 USDC
4. 利润：281 USDC (2.81%)

执行要点

  1. 速度第一：用 Flashbots 避免抢跑
  2. Gas 优化：选择低峰时段
  3. 滑点控制：设置合理上限
  4. 监控工具：自建价格监控脚本

风险

  • 价格快速变化
  • 交易失败损失 Gas
  • MEV 机器人竞争

流动性管理策略 - 从被动 LP 到主动管理

三种 LP 策略对比

策略难度收益潜力风险适合人群
被动 LP⭐⭐⭐⭐⭐新手
主动调仓⭐⭐⭐⭐⭐⭐⭐⭐⭐进阶
集中流动性⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐专业

策略一：被动 LP

操作方式

  1. 存入 50:50 双币
  2. 获得 LP Token
  3. 坐收手续费，不做任何操作

优点

  • 简单，一次设置即可
  • 无需关注市场波动
  • 适合长期看好双币

缺点

  • 资金效率低
  • 无常损失完全暴露
  • APY 通常较低（5-15%）

推荐场景

  • 稳定币对（USDC/DAI）
  • 强相关资产（ETH/stETH）
  • 长期持有信念强

策略二：主动调仓

核心思路

定期 rebalance 回 50:50 比例，降低无常损失。

操作步骤

1. 监控池子比例
2. 当偏离>20% 时触发
3. 卖出多的，买入少的
4. 重新存入池子

优点

  • 降低无常损失 30-50%
  • 可在震荡市获利
  • 灵活性高

缺点

  • 需要频繁操作
  • 产生额外 gas 费
  • 可能错过单边行情

工具推荐

  • DeFi Saver：自动 rebalance
  • Gelato：条件触发执行
  • 自建脚本监控

革命性创新

V3 允许自定义价格区间，资金效率提升 10-100 倍。
