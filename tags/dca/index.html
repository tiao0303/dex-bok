限价单交易

什么是限价单？

市价单：立即以当前价格成交。 限价单：设定目标价格，达到时自动成交。

为什么需要限价单？

  • 不需要时刻盯盘
  • 避免情绪化交易
  • 可以设置止盈止损
  • 更好的价格执行

设置限价单步骤（以 1inch 为例）

  1. 打开 https://app.1inch.io
  2. 连接钱包
  3. 选择交易对
  4. 点击"Limit Order"
  5. 设置：
    • 价格：你希望的成交价
    • 数量：交易金额
    • 有效期：订单过期时间
  6. 签名订单（不需要 Gas）
  7. 等待成交

限价单策略

买入策略：

当前价格：$100
目标买入价：$80（-20%）
设置：当价格跌到$80 时自动买入

止盈策略：

持仓成本：$100
目标卖出价：$150（+50%）
设置：当价格涨到$150 时自动卖出

止损策略：

持仓成本：$100
止损价：$80（-20%）
设置：当价格跌到$80 时自动卖出
