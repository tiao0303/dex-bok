T1 新手入门 03 - 第一笔交易：从充值到完成兑换

准备好了吗？这是你第一次在 DEX 上交易的完整教程！

准备工作

必需物品

  1. ✅ 已创建的钱包（参考 T1-02）
  2. ✅ 钱包里有 Gas 费（ETH/BNB 等）
  3. ✅ 想要交易的 USDT/USDC
  4. ✅ 知道要交易的代币合约地址

第一步：获取 Gas 费

DEX 交易需要支付链上 Gas 费，不同链需要不同的原生代币：

链Gas 代币建议准备
EthereumETH$50-100
BSCBNB$20-50
ArbitrumETH$10-20
SolanaSOL$5-10
PolygonMATIC$5-10

如何获取？

方法 1：从 CEX 提现（推荐新手）

  1. 在币安/OKX 等交易所购买 ETH/BNB
  2. 提现到你的钱包地址
  3. 选择正确的网络（ERC20/BEP20）

⚠️ 注意：提现地址必须是你钱包的地址，网络必须匹配！

方法 2：使用跨链桥

如果你在其他链有资产，可以使用跨链桥：

  • Stargate
  • Hop Protocol
  • LayerZero

方法 3：朋友转账

让有 crypto 的朋友直接转给你一些 Gas 费。

第二步：访问 DEX

选择 DEX

根据你的链选择：

链推荐 DEX网址
EthereumUniswaphttps://app.uniswap.org
BSCPancakeSwaphttps://pancakeswap.finance
ArbitrumUniswap/Camelothttps://app.uniswap.org
SolanaJupiterhttps://jup.ag

⚠️ 安全警告：

  • 手动输入网址，不要点 Google 广告
  • 收藏官方网址
  • 检查网址是否正确（钓鱼网站层出不穷）

连接钱包

  1. 打开 DEX 网站
  2. 点击右上角"Connect Wallet"
  3. 选择你的钱包（MetaMask/Phantom 等）
  4. 在钱包弹窗中点击"连接"
✅ 连接成功标志：右上角显示你的地址（如 0x742d...f0bEb）

第三步：执行兑换

界面说明

以 Uniswap 为例：

┌─────────────────────────────────┐
│  From: [ETH ▼]  [输入数量]      │
│         ↓                       │
│  To:   [USDC▼]  [预计收到]      │
│                                 │
│  [滑点设置] [价格影响]          │
│                                 │
│  [Swap] 按钮                    │
└─────────────────────────────────┘

操作步骤

  1. 选择 From 代币：点击"ETH"，选择你要卖出的代币
  2. 输入数量：输入你想交易的金额
  3. 选择 To 代币：点击"Select token"，选择你想买的代币
  4. 查看报价：
    • 汇率：1 ETH = ? USDC
    • 价格影响：你的交易对价格的影响
    • 费用：网络费 + 平台费
  5. 设置滑点（如果需要）：点击设置图标，一般 0.5%-1%
  6. 点击 Swap：确认交易详情
  7. 在钱包中确认：MetaMask 会弹出确认窗口
  8. 等待确认：交易上链需要几秒到几分钟

理解交易参数

滑点（Slippage）

定义：实际成交价与预期价格的偏差。

为什么需要？因为价格在你确认交易时可能已经变化。

设置建议：

代币类型滑点设置
主流币（ETH/USDC）0.5%
小市值代币1-3%
土狗币/Meme5-15%（风险高）

⚠️ 警告：滑点设置过高可能被 MEV 机器人攻击！

价格影响（Price Impact）

定义：你的交易量占流动性池的比例。

建议：

  • < 1%：安全
  • 1-3%：可以接受

  • 3%：考虑分批交易

Gas 费（网络费）

组成：

  • Base Fee：基础费用（燃烧掉）
  • Priority Fee：优先费（给矿工）

查看 Gas：

  • Etherscan Gas Tracker
  • Gas Now

省钱技巧：

  • 避开高峰时段（美国工作时间）
  • 使用 Layer 2（Arbitrum/Optimism）
  • 选择低 Gas 链（BSC/Solana）

第四步：确认交易

交易状态

  1. Pending：交易已提交，等待确认
  2. Confirmed：交易成功，上链确认
  3. Failed：交易失败（Gas 费不退）

查看交易

点击交易哈希（Tx Hash），在区块浏览器查看：

  • Ethereum: https://etherscan.io
  • BSC: https://bscscan.com
  • Solana: https://solscan.io

验证代币到账

  1. 打开钱包
  2. 点击"导入代币"
  3. 粘贴代币合约地址
  4. 确认余额

常见问题

Q1: 交易一直 Pending 怎么办？

原因：Gas 费太低，矿工不打包。

解决：

  • 等待（可能最终会确认）
  • 加速交易（MetaMask 有"Speed Up"选项）
  • 取消交易（需要再付一次 Gas）

Q2: 交易 Failed 了？

常见原因：

  • 滑点设置太低
  • 流动性不足
  • 代币有交易限制
  • Gas 费不足

解决：检查错误信息，调整参数重试。

Q3: 收到的代币数量不对？

可能原因：

  • 价格波动
  • 滑点设置
  • 代币有交易税（某些 Meme 币有 10% 税）

Q4: 钱包显示"用户拒绝"？

你取消了交易，不会有任何损失（除了可能的 Gas）。

交易检查清单

在点击 Swap 前，确认：

  • 网址正确（不是钓鱼网站）
  • 连接的钱包正确
  • From/To 代币正确
  • 数量正确
  • 滑点设置合理
  • Gas 费充足
  • 价格影响可接受
  • 合约地址已验证
