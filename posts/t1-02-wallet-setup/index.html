T1 新手入门 02 - 创建钱包：安全保管你的加密资产

在开始使用 DEX 之前，你需要一个加密钱包。这是你在 Web3 世界的身份和保险箱。

什么是加密钱包？

加密钱包不是"装"币的容器，而是管理私钥的工具：

  • 公钥 → 你的地址（可以公开，类似银行账号）
  • 私钥 → 你的密码（绝对不能泄露，类似银行卡密码）
  • 助记词 → 私钥的人类可读形式（12-24 个单词）
助记词 → 私钥 → 公钥 → 地址

钱包类型对比

🔥 热钱包（Hot Wallet）

联网的钱包，方便但安全性较低。

钱包平台适合
MetaMask浏览器插件/手机Ethereum 及 EVM 链
Phantom浏览器/手机Solana
Trust Wallet手机多链支持
Rainbow手机/浏览器Ethereum，UI 友好

🧊 冷钱包（Cold Wallet）

离线存储，安全性最高。

钱包价格特点
Ledger Nano X~$149蓝牙，支持多链
Trezor Model T~$219触摸屏，开源
OneKey~$119性价比高

建议：新手从热钱包开始，资产超过$1000 后考虑冷钱包。

创建钱包：MetaMask 详细教程

步骤 1：下载安装

  1. 访问官网：https://metamask.io
  2. 点击"Download"
  3. 选择你的浏览器（Chrome/Edge/Brave）
  4. 添加到浏览器

⚠️ 警告：只从官网下载！不要点击 Google 广告！

步骤 2：创建钱包

  1. 点击"Create a new wallet"
  2. 设置强密码（12 位以上，含大小写和符号）
  3. 备份助记词（最重要的一步！）

步骤 3：备份助记词

⚠️ 助记词示例（不要使用这个！）：

witch collapse practice feed shame open 
despair creek road again ice least

安全备份规则：

  1. ✅ 手写在纸上，存放在安全地方
  2. ✅ 使用金属助记词板（防火防水）
  3. ✅ 可以分多处存放（如 3-2 原则：3 份中任意 2 份可恢复）
  4. ❌ 不要截图保存
  5. ❌ 不要存在云盘/笔记软件
  6. ❌ 不要告诉任何人
  7. ❌ 不要输入到任何网站

步骤 4：验证助记词

MetaMask 会让你按顺序点击助记词，确保你正确备份。

步骤 5：完成！

现在你有了一个钱包地址，类似：

0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb

钱包安全最佳实践

🛡️ 必做事项

  1. 助记词离线保存：只写在纸上或金属板上
  2. 使用硬件钱包：大额资产用冷钱包
  3. 分离钱包：日常交易和长期存储分开
  4. 启用额外密码：MetaMask 可设置额外密码
  5. 定期检查：确认助记词可恢复

🚫 绝对禁止

  1. 把助记词存在手机/电脑里
  2. 截图或拍照助记词
  3. 用邮箱/云盘备份助记词
  4. 在任何网站输入助记词
  5. 相信"客服"索要助记词（100% 是骗子）

🔍 识别钓鱼

多链钱包配置

添加 BSC 网络到 MetaMask

  1. 打开 MetaMask
  2. 点击网络下拉 → “Add Network”
  3. 填写：
    • Network Name: BNB Smart Chain
    • RPC URL: https://bsc-dataseed.binance.org
    • Chain ID: 56
    • Symbol: BNB
    • Block Explorer: https://bscscan.com

添加其他网络

使用 Chainlist.org 一键添加。

常见错误

❌ 错误 1：助记词输错顺序

助记词顺序非常重要！witch collapse practice 和 practice collapse witch 是完全不同的钱包。

❌ 错误 2：丢失助记词

没有助记词 = 永久丢失资产，没有任何人可以帮你恢复。

❌ 错误 3：转错链

把 ETH 转到 BSC 地址？资产还在，但需要跨链桥找回。

检查清单

在开始交易前，确认：

  • 助记词已手写备份
  • 助记词存放在安全地方
  • 记住了钱包密码
  • 知道如何查看地址
  • 了解如何切换网络
  • 准备了 Gas 费（ETH/BNB 等）
