流动性管理策略 - 从被动 LP 到主动管理

孙洋

March 18, 2026

三种 LP 策略对比

策略难度收益潜力风险适合人群
被动 LP⭐⭐⭐⭐⭐新手
主动调仓⭐⭐⭐⭐⭐⭐⭐⭐⭐进阶
集中流动性⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐⭐专业

策略一：被动 LP

操作方式

  1. 存入 50:50 双币
  2. 获得 LP Token
  3. 坐收手续费，不做任何操作

优点

  • 简单，一次设置即可
  • 无需关注市场波动
  • 适合长期看好双币

缺点

  • 资金效率低
  • 无常损失完全暴露
  • APY 通常较低（5-15%）

推荐场景

  • 稳定币对（USDC/DAI）
  • 强相关资产（ETH/stETH）
  • 长期持有信念强

策略二：主动调仓

核心思路

定期 rebalance 回 50:50 比例，降低无常损失。

操作步骤

1. 监控池子比例
2. 当偏离>20% 时触发
3. 卖出多的，买入少的
4. 重新存入池子

优点

  • 降低无常损失 30-50%
  • 可在震荡市获利
  • 灵活性高

缺点

  • 需要频繁操作
  • 产生额外 gas 费
  • 可能错过单边行情

工具推荐

  • DeFi Saver：自动 rebalance
  • Gelato：条件触发执行
  • 自建脚本监控

策略三：集中流动性（Uniswap V3）

革命性创新

V3 允许自定义价格区间，资金效率提升 10-100 倍。

区间选择策略

区间宽度资本效率被甩出风险适合场景
±10%100x高稳定币
±30%30x中主流币
±100%5x低长尾资产

实战案例：ETH/USDC

当前价格：1800 USDC

保守策略：

  • 区间：1500 - 2100（±17%）
  • 效率：约 20x
  • 风险：中等

激进策略：

  • 区间：1700 - 1900（±5%）
  • 效率：约 80x
  • 风险：高，易被甩出

被甩出后怎么办？

  1. 等待回归：价格回到区间内
  2. 调整区间：重新设置（需 gas）
  3. 退出 LP：转换为持有

策略选择建议

新手（<1 万 USD）

→ 被动 LP，先熟悉机制

进阶（1-10 万 USD）

→ 主动调仓 + 部分 V3 集中流动性

专业（>10 万 USD）

→ 多策略组合，对冲风险

风险管理

  1. 不要 All-in：最多 50% 资金做 LP
  2. 分散池子：3-5 个不同池子
  3. 定期复盘：每周检查一次
  4. 设置止损：亏损>20% 考虑退出
