收入分析价值

核心指标详解

1. 收入（Revenue）⭐⭐⭐⭐⭐

定义

协议收入 = 用户支付费用 - 流动性提供者分成

计算公式

协议收入 = 总费用 × 协议分成比例

示例（Uniswap V3）：
总费用 = $100 万/天
LP 分成 = 100%
协议收入 = $0（V3 无协议分成）

示例（Aave）：
总利息 = $50 万/天
LP 分成 = 70%
协议收入 = $15 万/天

对比分析

协议日收入月收入年收入（年化）
Uniswap$0$0$0
Aave$15 万$450 万$5475 万
MakerDAO$20 万$600 万$7300 万
Curve$5 万$150 万$1825 万

2. 费用（Fees）⭐⭐⭐⭐

定义

总费用 = 用户为使用协议支付的全部费用

与收入的区别

费用 = 收入 + LP 分成

示例（Curve）：
用户支付费用 = $10 万/天
LP 分成 = 50% = $5 万
协议收入 = $5 万

意义

  • 费用反映需求强度
  • 收入反映盈利能力
  • 两者结合看商业模式

3. P/S 比率 ⭐⭐⭐⭐⭐

定义

P/S = 市值 / 年收入

计算示例

Aave:
市值 = $15 亿
年收入 = $5475 万
P/S = 15 / 0.5475 = 27.4

估值参考

P/S估值水平操作建议
<5低估买入 ✅
5-15合理持有
15-30偏高谨慎
>30高估卖出 ⚠️

行业对比

协议P/S评价
MakerDAO8.2低估 ✅
Aave27.4合理
UniswapN/A无收入
Lido45.3高估 ⚠️

4. P/E 比率 ⭐⭐⭐⭐

定义

P/E = 市值 / 年利润

DeFi 特殊性

利润 = 收入 - 代币释放

示例：
协议收入 = $1000 万/年
代币释放 = $1500 万/年
利润 = -$500 万（亏损）

解读

  • P/E < 20：盈利能力强
  • P/E 负值：代币释放>收入
  • 高通胀协议慎用 P/E

5. FDV/TVL ⭐⭐⭐⭐

定义

FDV/TVL = 全流通市值 / 总锁仓价值

意义

  • <1：可能被低估
  • 1-3：合理区间

  • 3：可能高估

对比

Uniswap$45 亿$40 亿1.13
Compound$8 亿$30 亿0.27 ✅

实战案例分析

案例一：发现 MakerDAO 低估

数据（2023 Q1）

市值：$12 亿
年收入：$1.5 亿
P/S：8.0
TVL：$60 亿
FDV/TVL：0.2

分析

1. P/S = 8 → 远低于行业平均 20
2. 收入稳定增长（DAI 需求增加）
3. TVL 持续上升
4. 代币释放少（通缩模型）

结论：严重低估
行动：买入 MKR
结果：3 个月 +80%

案例二：识别 Lido 高估

数据（2023 Q2）

市值：$30 亿
年收入：$5000 万
P/S：60
TVL：$200 亿
FDV/TVL：0.15

分析

1. P/S = 60 → 极高
2. 收入依赖 ETH 质押
3. 竞争加剧（Rocket Pool 等）
4. 代币释放压力大

结论：高估风险
行动：观望/减仓
结果：后续下跌 40%

建立估值模型

简化 DCF 模型

公式

合理市值 = 年收入 × 合理 P/S

示例（Aave）：
年收入 = $5475 万
合理 P/S = 15（行业平均）
合理市值 = 5475 万 × 15 = $8.2 亿

当前市值 = $15 亿
→ 高估 83%

敏感性分析

不同 P/S 假设：
P/S = 10 → 市值 $5.5 亿
P/S = 15 → 市值 $8.2 亿
P/S = 20 → 市值 $11 亿
P/S = 25 → 市值 $13.7 亿

相对估值法

同行对比

选择可比协议：
- Aave、Compound、MakerDAO

计算平均 P/S：
(27.4 + 15.2 + 8.2) / 3 = 16.9

应用：
目标协议收入 × 16.9 = 合理市值

我的投资框架

第一步：筛选

条件：
- P/S < 15
- 收入增长 >30%
- TVL 稳定
- 代币释放<收入

结果：3-5 个候选

第二步：深入分析

每个候选：
1. 阅读文档和治理提案
2. 分析竞争格局
3. 评估团队背景
4. 检查智能合约审计

第三步：建仓

资金分配：
- 第一候选：40%
- 第二候选：30%
- 第三候选：20%
- 现金：10%

第四步：监控

每周检查：
- 收入变化
- TVL 趋势
- 竞争动态
- 治理提案

触发条件：
- 收入连续 2 月下降 → 减仓
- P/S > 30 → 考虑卖出
- 发现更好机会 → 调仓

总结

协议收入分析核心：

  1. 关注真实收入：而非代币价格
  2. 用 P/S 估值：DeFi 版 P/E
